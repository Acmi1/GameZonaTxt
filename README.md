Установка на Termux(Android):
1. Download Termux
2. apt update && upgrade
3. pkg install python3
4. pkg install pip
5. pkg install git
6. git clone https://github.com/Acmi1/GameZonaTxt.git
Запуск игры:
1. cd GameZoneTxt
2. chmod +x *
3. start.py
